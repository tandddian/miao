# 数组的基本介绍与使用；leetcode介绍

## 数组的基本操组

```
a.push(5): 向末尾增加一个元素，push 返回的是数组的新长度，而且它可以一次性 push 多个元素

a.pop(): 删除末尾元素并返回, pop一次只能删除一个元素

a.shift(): 从前面删除并返回一个元素

a.unshift(val1, val2): 从前面添加元素 

```

二维数组

```
var a = [[1, 3, 4], [2, 6, 9]]

a[0][2] // 4
a[1] // [2, 6, 9]

```
